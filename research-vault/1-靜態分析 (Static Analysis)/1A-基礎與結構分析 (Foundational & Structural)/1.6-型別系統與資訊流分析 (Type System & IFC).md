# 1.6 型別系統與資訊流分析 (Type System / Information Flow Typing)

## 📌 核心精神 (Core Concept)
透過靜態型別系統，強制標注資料的安全等級（High / Low），編譯期驗證高安全性資料不會「流出」到低安全性輸出（Non-interference 保證）。

## 🔍 子技術流派 (Sub-Genres)
### Security Type Systems
- **簡介**: 資訊流安全性檢查與標注。代表工具：Jif、FlowSpec。


## 🏷️ 學術常用關鍵字 (Keywords)
`non-interference`, `security type`, `label propagation`, `information flow control (IFC)`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
`Jif`, `FlowSpec`
