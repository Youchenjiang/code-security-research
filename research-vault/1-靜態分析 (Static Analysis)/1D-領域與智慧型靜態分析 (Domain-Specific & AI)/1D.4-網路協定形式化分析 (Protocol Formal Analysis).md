# 1D.4 網路協定形式化分析 (Protocol Formal Analysis)

## 📌 核心精神 (Core Concept)
在協定規格與設計層面，使用代數或符號模型抽象化加密協定的訊息交換流程，在 Dolev-Yao 攻擊者模型下證明或尋找攻擊路徑。

## 🔍 子技術流派 (Sub-Genres)
### Symbolic Protocol Verification (符號協定驗證)
- **簡介**: 將密碼協定抽象為代數項重寫系統以自動驗證。代表工具：ProVerif、Tamarin Prover。

### Computational Model Verification (計算模型驗證)
- **簡介**: 在機率多項式時間內證明協定安全性。代表工具：CryptoVerif。


## 🏷️ 學術常用關鍵字 (Keywords)
`cryptographic protocol verification`, `Dolev-Yao model`, `Tamarin Prover`, `ProVerif`, `symbolic protocol analysis`, `secrecy properties`, `authentication logic`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
