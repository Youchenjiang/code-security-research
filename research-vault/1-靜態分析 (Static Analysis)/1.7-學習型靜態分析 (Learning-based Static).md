# 1.7 基於機器學習的靜態檢測 (Learning-based Static Analysis)

## 📌 核心精神 (Core Concept)
以原始碼或程式中間表示作為模型輸入，訓練神經網路自動學習漏洞模式，免去人工撰寫規則。近年發展為最活躍的子領域。

## 🔍 子技術流派 (Sub-Genres)
### 序列模型 (Sequence-based)
- **簡介**: 將程式碼視為 token 序列，使用 LSTM / Transformer 分類。代表工作：VulDeePecker、SySeVR、LineVul。

### 圖神經網路 (GNN-based)
- **簡介**: 將 CPG / PDG / AST 作為圖輸入，以 GNN 學習結構化語意。代表工作：Devign、Reveal、DeepWukong、LLMxCPG。

### 預訓練語言模型 (Pre-trained LM)
- **簡介**: 使用 CodeBERT / UniXcoder / CodeT5 等在大量程式碼上預訓練，再微調於漏洞偵測。代表工作：VulBERTa、LineVul、AugSliceVul。

### LLM 輔助靜態分析 (LLM + Static)
- **簡介**: 以 LLM 自動推斷 taint 規格（Source/Sink）或補充靜態分析的語意理解，再交回傳統工具做嚴謹驗證。代表工作：IRIS、GPTScan。
