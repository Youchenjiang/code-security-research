# 1.3 抽象解釋 (Abstract Interpretation)

## 📌 核心精神 (Core Concept)
以數學上「正確近似（Sound Approximation）」的方式，計算程式所有可能執行狀態的超集合（Over-approximation），保證不漏報（Sound）。

## 🔍 子技術流派 (Sub-Genres)
### 數值域抽象 (Numerical Abstract Domains)
- **簡介**: 追蹤整數範圍、陣列邊界，偵測 Buffer Overflow、整數溢位。代表工具：Astrée、Polyspace、Frama-C。

### 記憶體安全分析
- **簡介**: 偵測 use-after-free、double-free、null pointer dereference。代表工具：Infer（基於雙模擬分離邏輯的 Bi-abduction）。


## 🏷️ 學術常用關鍵字 (Keywords)
`interval domain`, `octagon domain`, `polyhedra domain`, `Galois connection`, `widening operator`, `separation logic`, `shape analysis`, `heap analysis`, `Bi-abduction`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
`Astrée`, `Polyspace`, `Frama-C (Value Analysis)`, `Infer`
