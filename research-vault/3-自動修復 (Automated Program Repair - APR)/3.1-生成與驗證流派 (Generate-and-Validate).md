# 3.1 生成與驗證流派 (Generate-and-Validate APR / Search-based APR)

## 📌 核心精神 (Core Concept)
以啟發式搜索或演化演算法在補丁空間中搜索，對每個候選補丁執行測試套件驗證，找到第一個通過測試的補丁即為輸出。是 APR 研究的起源，開創了整個領域。

## 🔍 子技術流派 (Sub-Genres)
### 遺傳程式設計基礎 APR (Genetic Programming)
- **簡介**: 以程式語句為「基因」，透過隨機刪除、插入、替換等操作生成候選補丁。代表工具：GenProg、MutRepair、SPR。

### 隨機/啟發式突變 APR
- **簡介**: 枚舉候選補丁，配合故障定位排序優先測試高可疑點。代表工具：RSRepair、ACS、jGenProg、Astor 框架。


## 🏷️ 學術常用關鍵字 (Keywords)
`genetic programming APR`, `search-based repair`, `plausible patch`, `fitness function`, `donor code`, `mutation-based repair`, `fault localization + repair`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
`GenProg`, `MutRepair`, `SPR`, `RSRepair`, `ACS`, `Astor`
