# 3B.3 範本 / 模式匹配流派 (Template-based / Pattern-based APR)

## 📌 核心精神 (Core Concept)
從歷史人類補丁中挖掘反覆出現的「修復模板」（如：加入 null check、修改條件式、插入邊界檢查），再將模板套用至對應的故障位置，模式可解釋性強。

## 🔍 子技術流派 (Sub-Genres)
### 通用 Bug 修復模板
- **簡介**: 聚焦於通用軟體缺陷，建立跨專案、跨類型的修復模板庫。代表工具：TBar、SimFix、AVATAR。

### 安全漏洞修復模板 (Vulnerability-Specific Patterns)
- **簡介**: 從歷史 CVE 補丁中挖掘安全相關修復模式，針對特定 CWE 類型設計修復模板。代表工具：AVATAR、FixMiner、VulKey。


## 🏷️ 學術常用關鍵字 (Keywords)
`fix pattern`, `template-based APR`, `code change pattern`, `donor code`, `fix pattern mining`, `security patch pattern`, `CVE fix template`, `CWE-specific repair`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
`TBar`, `SimFix`, `AVATAR`, `FixMiner`, `VulKey`
