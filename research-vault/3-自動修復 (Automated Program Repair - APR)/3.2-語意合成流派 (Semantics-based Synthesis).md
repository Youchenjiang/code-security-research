# 3.2 語意合成流派 (Semantics-based / Synthesis-based APR)

## 📌 核心精神 (Core Concept)
將修復問題轉化為約束求解（Constraint Solving）問題，以 SMT 求解器或符號執行精確合成出語意正確的修復表達式，理論上比搜索流派更嚴謹。

## 🔍 子技術流派 (Sub-Genres)
### 基於 SMT 求解的修復合成
- **簡介**: 從通過測試的輸入/輸出對提取語意約束，形式化為 SMT 公式，求解器合成修復的條件式或表達式。代表工具：Nopol、Angelix、S3、SemFix。

### 基於符號執行的合成
- **簡介**: 以符號執行探索觸發 bug 的路徑，精確識別修復位置，並以程式合成填補修復邏輯。代表工具：ExtractFix、SymFix。


## 🏷️ 學術常用關鍵字 (Keywords)
`synthesis-based APR`, `SMT solver repair`, `SemFix`, `Nopol`, `Angelix`, `angelic fix derivation`, `extraction-based repair`, `repair constraint synthesis`

## 🛠️ 代表性工具 / 框架 (Representative Tools)
`Nopol`, `Angelix`, `SemFix`, `ExtractFix`, `SymFix`
